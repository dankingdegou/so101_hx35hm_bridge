# HX35HM + SO101 Control Stack

This repository contains the control-side stack for the HX35HM + SO101 arm setup.

It is meant to help you bring up and use the robot control pipeline:

- HX35HM bridge and servo control
- ROS 2 bringup and controllers
- MoveIt planning and execution
- teleop
- hand-eye and camera calibration support
- grasping workflow tools and notes

## Start here

If you want the shortest path to running MoveIt with the real arm, read:

- [HX35HM_SO101_MoveIt规划控制启动流程.md](src/HX35HM_SO101_MoveIt规划控制启动流程.md)
