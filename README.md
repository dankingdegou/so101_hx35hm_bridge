# SO101 HX35HM Control Export

This repository is a cleaned export of the HX35HM + SO101 arm control stack.

It contains the robot-control-related workspace pieces I prepared locally:

- `src/so101_hx35hm_bridge`
- selected control packages from `so101-ros-physical-ai`
- HX35HM assembly / calibration / MoveIt workflow notes

The goal is to keep the arm control code in one place for the target GitHub repo.
